\[\widehat{\beta} \sim N \Big( \beta ,\sigma^{2} (X‚Ä≤X)^{‚àí1} \Big)\] Esto implica que los estimadores \(\widehat{\beta_{i}}\) son:

Inferencia sobre los coeficientes

Si los \(\widehat{\beta}_{i}\) se distribuyen normal, entonces:

\[ {T_j} = {\frac{\widehat\beta_j - \beta_j}{\text{se}\left(\widehat\beta_j\right)} \sim t_{n - p}}, \hspace{1cm} j = 0, 1, \ldots, k \] y

El intervalo de confianza para \(\beta_{i}\) esta dado por :

\[ \widehat\beta_j \pm t_{\alpha/2, n - p}\, \text{se}\left(\widehat\beta_j\right) \]

y el estadistico de prueba para \(Ho: \beta_{j} = B_{0}\) vs \(Ho: \beta_{j} \neq B_{0}\) es

\[T_{j,0} = \dfrac{\widehat\beta_j - B_{j,0}}{\text{se}\left(\widehat\beta_j\right)} \overset{\text{bajo }H_0}{\sim} t_{v: n - p}\]

En ocasiones es necesario realizar pruebas de hip√≥tesis sobre un subconjunto de par√°metros, en estos caso se utiliza la prueba F, comparando el modelo completo contra el modelo reducido.
